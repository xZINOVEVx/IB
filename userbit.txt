Спецальные биты управления доступом
1. userbit - утсанавливается только для файлов 
Позволяет запустить исполняемый файл от имени владельца
Необходим для программ выполняющихся от имени обычных пользователей, которые изменяют объекты файловой системы, защищенные от записи для всех категорий пользователей.

2. Установка userbit:(в режиме администратора):
sudo chmod u+s <имя исполняемой программы>
sudo chmod u+s passwd

3.Вывод команды stat для программы passwd:
ZINOVEV@alexandr-Aspire-Z5751:~/IB$ stat /bin/passwd
  Файл: /bin/passwd
  Размер: 64152         Блоков: 128        Блок В/В: 4096   обычный файл
Устройство: 8/4 Инода: 3150126     Ссылки: 1
Доступ: (4755/-rwsr-xr-x)  Uid: (    0/    root)   Gid: (    0/    root)

4.В случае установки s-бита владельца для файла, у которого нет прав доступа на выполнение, s-бит выводиться в верхнем регистре
stat /home/student/1.txt
Access: (4664/ -rwSrw-r--)


5. Сброс user-bit в режиме администратора:
sudo chmod u-s <имя исполняемой программы>
